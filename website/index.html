Meu primeiro projeto Docker!

Projeto desenvolvido no curso de Docker na DIO

Criando uma imagem docker para um container executando um servidor apache.


Etapas:

  1. Instalar o Docker Compose: $ apt install docker-compose
  2. Gerar o arquivo: $ docker-compose up -d
  3. Verificar se já está rodando: $ docker container ls
  4. Abrir o site com o ip da imagem na porta 80
